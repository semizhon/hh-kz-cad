üìã About This API

This FastAPI application searches for CAD-related jobs in Kazakhstan using the HeadHunter API. It specifically looks for jobs requiring skills in:

  • AutoCAD
  • Revit
  • Inventor
  • Fusion 360
  • Advance Steel
  • And other CAD software

üîó API Endpoints

GET /

Web Interface - Interactive web interface for searching jobs

GET /jobs

API Endpoint - JSON API for searching jobs

üìù Usage Examples

Basic Search

curl "https://your-api-url.com/jobs?keywords=AutoCAD&country=Kazakhstan"

Advanced Search with Multiple Keywords

curl "https://your-api-url.com/jobs?keywords=AutoCAD&keywords=Revit&country=Kazakhstan&pages=2&per_page=50"

Filter by Specific Products

curl "https://your-api-url.com/jobs?keywords=CAD&products=Fusion 360&products=Inventor&country=Kazakhstan"

üîß Parameters

  • keywords - Search keywords (default: AutoCAD,Revit,Inventor,Fusion 360,Fusion,Advance Steel)
  • country - Country to search in (default: Kazakhstan)
  • pages - Number of pages to search (default: 1)
  • per_page - Results per page (default: 100)
  • products - Filter by specific products
